# AGRONITI
Smart AI-Powered Agri-Market Ecosystem

AGRONITI is a low-bandwidth, AI-integrated agricultural marketplace prototype designed to connect Farmers, Buyers, and Government/Admin officials in a transparent and data-driven ecosystem.

It helps reduce post-harvest losses, prevent price scams, and increase farmer income through smart decision support tools.
